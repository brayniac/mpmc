## Usage

To use `mpmc`, first add this to your `Cargo.toml`:

```toml
[dependencies]
mpmc = "*"
```

Then, add this to your crate root:

```rust
extern crate mpmc;
```

The API documentation of this library can be found at
[docs.rs/mpmc](https://docs.rs/mpmc/).
